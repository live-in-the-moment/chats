chats-基于Linux 的多用户实时聊天程序
1. 基于Linux系统开发、使用C语言、Socket编程、Tcp协议实现多个客户端之间的局域网通信;
2. 程序采用C/S架构。设立一台服务器PC，客户端PC 通过服务器PC，实现多客户端主机相互通信的功能;
3. 目前已经实现了：
    - 注册;
    - 登录;
    - 私聊模式;
    - 群聊模式;
    - 查询在线用户;
    - 查询私聊模式的聊天记录;
    - 查询群聊模式的聊天记录;
    - 客户端之间传输文件;
    - 心跳监听;
4. 数据存储使用了sqlite数据库;
5. 具体功能实现看功能介绍文档;
